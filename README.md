# 🪶 EvilFont

A Magisk/KernelSU module to change your Arabic/Persian font to Teshrin + IOS emojis 

## 📜 Instruction

- Download EvilFont zip file
- Go to Magisk/KernelSU module manager
- Install zip file
- Reboot and Enjoy
